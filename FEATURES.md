# Compare - 기능 명세서

## 구현 완료 기능 (전체 40개)

### 텍스트 비교 (TextCompareWidget)
- [x] **1** 두 파일 나란히 표시 (side-by-side diff)
- [x] **2** 라인 단위 diff 색상 강조 (삽입/삭제/변경/동일)
- [x] **3** 라인 번호 표시 (LineNumberArea 사이드 위젯)
- [x] **4** 다음/이전 변경사항 이동 (`F7` / `Shift+F7`)
- [x] **5** 현재 변경사항 블록 강조 (오버레이)
- [x] **6** 변경사항 카운터 표시 ("N / M")
- [x] **7** 좌우 패널 스크롤 동기화
- [x] **8** 파일 열기 버튼 (각 패널별)
- [x] **17** 인라인 편집 기능 (편집 모드 토글)
- [x] **18** 변경사항 좌↔우 복사 버튼 ("복사 →" / "← 복사")
- [x] **20** 저장 기능 — Ctrl+Shift+S
- [x] **21** Undo/Redo — QPlainTextEdit 내장 지원
- [x] **22** 텍스트 검색 — Ctrl+F (검색바 슬라이드인)
- [x] **23** 정규식 검색 (검색바 "정규식" 체크박스)
- [x] **27** 문자(character) 단위 인라인 diff 강조
- [x] **28** 단어(word) 단위 diff 옵션 (툴바 콤보박스)
- [x] **29** diff 통계 표시 (추가/삭제/변경/동일 줄 수)
- [x] **36** 드래그 앤 드롭으로 파일 열기

### 폴더 비교 (FolderCompareWidget)
- [x] **9** 두 폴더 트리 비교 (파일 상태별 색상/아이콘)
- [x] **10** 파일 상태 표시 (동일/다름/좌측만/우측만)
- [x] **11** 재귀적 서브폴더 스캔
- [x] **12** 폴더 상태 자동 집계
- [x] **13** 파일 더블클릭 → 텍스트 비교 탭 오픈
- [x] **24** 폴더 비교 필터 (패턴, e.g. `*.log *.tmp`)
- [x] **25** 동일 파일 숨기기 옵션 (체크박스)
- [x] **26** 특정 확장자/패턴 무시 설정 (fnmatch 기반)
- [x] **36** 드래그 앤 드롭으로 폴더 열기

### 메인 윈도우 (MainWindow)
- [x] **14** 탭 기반 다중 비교 세션
- [x] **15** 메뉴바 및 단축키
- [x] **16** 다크 테마 UI (기본)
- [x] **34** 세션 저장/불러오기 — Ctrl+Shift+W
- [x] **37** 커맨드라인 인자 지원 — `python main.py file1 file2`
- [x] **38** 라이트 테마 지원 (보기 메뉴 > 라이트 테마)
- [x] **39** 탭 이름 변경 (탭 더블클릭 또는 보기 > 탭 이름 변경)
- [x] **40** 창 크기/위치 레이아웃 저장 (QSettings, 종료 시 자동 저장)
- [x] 최근 비교 목록 (파일 > 최근 비교)
- [x] **36** 전역 드래그 앤 드롭 (파일 타입 자동 감지)

### 3-Way Merge (MergeView)
- [x] **19** 3-way merge (Base / My / Their / 결과 4패널)
- [x] 충돌(CONFLICT) 자동 감지 및 강조
- [x] "My 채택" / "Their 채택" 전체 해결 버튼
- [x] 결과 편집 후 저장

### 이미지 비교 (ImageCompareWidget)
- [x] **32** 이미지 파일 비교 (픽셀 diff 시각화)
- [x] 픽셀 diff 통계 (다른 픽셀 수, 비율)
- [x] 확대/축소 슬라이더
- [x] 드래그 앤 드롭

### 바이너리 비교 (BinaryCompareWidget)
- [x] **33** 바이너리 파일 Hex dump 비교
- [x] 다른 행 강조 (노랑)
- [x] 크기/다른 행 통계
- [x] 드래그 앤 드롭

### Git 연동 (GitView)
- [x] **30** Git 저장소 열기
- [x] 브랜치 목록 조회
- [x] 커밋 목록 조회
- [x] 두 커밋 간 변경 파일 목록
- [x] 파일 선택 → 커밋 간 텍스트 diff

### SFTP 원격 비교 (FTPDialog)
- [x] **31** SFTP 연결 (호스트/포트/사용자/비밀번호/키 파일)
- [x] 원격 파일 브라우저 (트리뷰)
- [x] 파일 선택 → 로컬 임시 다운로드 → 텍스트 비교

### 설정 (SettingsDialog)
- [x] **35** 설정 화면 (테마, 폰트, Diff 색상)
- [x] 설정 영구 저장 (QSettings)

## 단축키 정리

| 단축키 | 기능 |
|--------|------|
| `Ctrl+T` | 새 텍스트 비교 탭 |
| `Ctrl+D` | 새 폴더 비교 탭 |
| `Ctrl+M` | 새 3-Way Merge 탭 |
| `Ctrl+I` | 새 이미지 비교 탭 |
| `Ctrl+B` | 새 바이너리 비교 탭 |
| `Ctrl+G` | Git 비교 탭 |
| `Ctrl+Q` | 앱 종료 |
| `F7` | 다음 변경사항 |
| `Shift+F7` | 이전 변경사항 |
| `Ctrl+F` | 텍스트 검색 |
| `Ctrl+Shift+S` | 왼쪽 파일 저장 |
| `Ctrl+Shift+W` | 세션 저장 |
| `Ctrl+,` | 설정 열기 |

## CLI 사용법

```bash
# 두 파일 비교
python main.py file1.txt file2.txt

# 일반 실행
python main.py
```

## diff 색상 코드 (다크 테마 기준)

| 상태 | 배경색 | 의미 |
|------|--------|------|
| `#1e1e1e` | 기본 | 동일 라인 |
| `#1a3d1a` | 초록 | 삽입 |
| `#3d1a1a` | 빨강 | 삭제 |
| `#3d3010` | 노랑 | 변경 |
| `#2a2a2a` | 회색 | 빈 줄 (패딩) |
| `#ffC800` + 투명도 | 골드 | 인라인 변경 강조 |
