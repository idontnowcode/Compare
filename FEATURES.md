# Compare - 기능 명세서

## 현재 구현된 기능

### 텍스트 비교 (TextCompareWidget)
- [x] 두 파일을 나란히 표시 (side-by-side diff)
- [x] 라인 단위 diff 계산 (`difflib.SequenceMatcher`)
- [x] diff 유형별 색상 강조
  - 동일 라인: 기본 배경 (`#1e1e1e`)
  - 삽입 (우측만 존재): 초록 (`#1a3d1a`)
  - 삭제 (좌측만 존재): 빨강 (`#3d1a1a`)
  - 변경 (양쪽 모두 다름): 노랑 (`#3d3010`)
  - 빈 줄 (패딩): 어두운 회색 (`#2a2a2a`)
- [x] 라인 번호 표시
- [x] 다음/이전 변경사항으로 이동 (`F7` / `Shift+F7`)
- [x] 현재 변경사항 강조 (흰색 오버레이)
- [x] 변경사항 카운터 표시 ("변경사항: 1 / 5")
- [x] 좌우 패널 스크롤 동기화
- [x] 파일 열기 버튼 (각 패널별)
- [x] UTF-8 인코딩 지원 (오류 무시 모드)

### 폴더 비교 (FolderCompareWidget)
- [x] 두 폴더의 파일/서브폴더 트리 비교
- [x] 파일 상태 표시
  - `=` 동일: MD5 해시 비교로 확인
  - `≠` 다름: 해시 불일치
  - `←` 좌측만: 우측 폴더에 없음
  - `→` 우측만: 좌측 폴더에 없음
- [x] 재귀적 서브폴더 스캔
- [x] 폴더 상태 자동 집계 (하위에 변경사항 있으면 `≠`)
- [x] 파일 더블클릭 → 텍스트 비교 탭 자동 오픈
- [x] 아이콘으로 파일/폴더 구분 (📄/📁)
- [x] 새로고침 버튼

### 메인 윈도우 (MainWindow)
- [x] 탭 기반 다중 비교 세션
- [x] 탭 닫기 버튼
- [x] 메뉴바 (파일 / 보기)
- [x] 새 텍스트 비교 탭 열기 (`Ctrl+T`)
- [x] 새 폴더 비교 탭 열기 (`Ctrl+D`)
- [x] 전역 단축키 (F7, Shift+F7)
- [x] 다크 테마 UI (VS Code 스타일)
- [x] 상태바

- [ ] 특정 확장자 무시 설정

- [ ] 창 분할 레이아웃 저장

## 단축키 정리

| 단축키 | 기능 |
|--------|------|
| `Ctrl+T` | 새 텍스트 비교 탭 |
| `Ctrl+D` | 새 폴더 비교 탭 |
| `Ctrl+Q` | 앱 종료 |
| `F7` | 다음 변경사항 |
| `Shift+F7` | 이전 변경사항 |
